# 2025 年计划

1. 总目标
- [ ] 事业：找到 3 人以上合伙创业，还清负债，存款达到 10w
- [ ] 身体：修身，身体能明确的感觉到精力充沛，并且身体健康，所有毛病去除。更显年轻，年年 18 岁
- [ ] 家人：家人身体健康
- [ ] 提升：古中医知识能做到小病不吃西药
- [ ] 提升：英语到能阅读英文文档的程度
- [ ] 提升：改变心理，去除自卑、内向、拖延症、懒
- [ ] 提升：抖音生活号视频总量达到 400 以上
- [ ] 提升：微信生活号好友到达 100 个
