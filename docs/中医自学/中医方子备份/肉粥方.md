# 肉粥方

> [微博正文](https://weibo.com/6980637174/4900344188110330) 

肉粥方（2人量）

配方：老藕100克 米100克  鮮薑9克 豬肉100克（有肥有瘦兼宜 部位不限）鹽 醬油（調味隨個人喜好）

做法：
1.蓮藕和薑剁碎備用  豬肉洗淨剁成肉末 米洗淨
2.鍋置火上 倒入少許油 將肉末倒入鍋中煎炒至香 倒入醬油上色 炒好後裝盤備用
3.鍋洗淨置火上 倒入1600毫升冷水 放藕、薑、米、豬肉，一同進去煮
4.大火煮開 轉小火40分鐘 煮至熟爛 放入鹽或調味即可食之

服用方法：早中晚皆可 或不拘時 7分飽

功：百姓日常良善飲膳 其味甚佳 為脾胃所喜 開胃進食 不滯胃 養心安神 主健脾補血  脾虛腹瀉 浮腫與體質虛羸者常服之能消腫長肉⋯
